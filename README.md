# IIoT Intrusion Detection System

## Description
This project implements an intrusion detection system using ESP32 and Blynk for real-time monitoring of IoT network activity.

## Technologies Used
- ESP32
- Blynk IoT Platform
- Arduino IDE
- WiFi Communication

## Features
- Detects unauthorized access in IoT network
- Sends alerts through Blynk dashboard
- Real-time monitoring

## Hardware Used
- ESP32
- Ultrasonic Sensor
- Resistors
